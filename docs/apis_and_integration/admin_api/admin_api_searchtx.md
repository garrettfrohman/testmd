# search

Search transactions for a merchant and its submerchants.

`GET/paymentiq/admin/v1/payments`

## Request Parameters

| Name                    | Located in | Description                                                                                                                                  | Schema   | Required |
| ----------------------- | ---------- | -------------------------------------------------------------------------------------------------------------------------------------------- | -------- | -------- |
| merchantId              | query      | merchantId                                                                                                                                   | string   | Yes      |
| currencies              | query      | Array of ISO 3-letter currency, e.g. SEK,EUR,USD                                                                                             | string   | No       |
| states                  | query      | Array of states, example SUCCESSFUL, REGISTERED                                                                                              | string   | No       |
| statuses                | query      | Array of statuses, example SUCCESS, WAITING_INPUT                                                                                            | string   | No       |
| txTypes                 | query      | Array of transaction types, example CreditcardDeposit, CreditcardWithdrawal                                                                  | string   | No       |
| psps                    | query      | Array of psps, example Entropay, Realex, Pacnet                                                                                              | string   | No       |
| initiatedPsps           | query      | Array of initiated psps, example Entropay, Realex, Pacnet                                                                                    | string   | No       |
| idList                  | query      | List of transaction ids                                                                                                                      | long     | No       |
| limit                   | query      | Limit on how many transactions to return. Max limit is 10 000. Default is 100.                                                               | integer  | No       |
| offset                  | query      | Offset useful for multi page results or pagination                                                                                           | integer  | No       |
| descOrder               | query      | Descending order                                                                                                                             | boolean  | No       |
| id                      | query      | Transaction id                                                                                                                               | long     | No       |
| originTxId              | query      | Origin transaction id                                                                                                                        | long     | No       |
| txRefId                 | query      | Normally (for now) the user is only allowed to search on the currently logged-in users mid. Transaction ref id in format ${merchantId}A${id} | string   | No       |
| pspStatusCode           | query      | Psp status code                                                                                                                              | string   | No       |
| pspAccount              | query      | Psp account                                                                                                                                  | string   | No       |
| channelId               | query      | Channel id                                                                                                                                   | string   | No       |
| merchantUserCat         | query      | Merchant user cat                                                                                                                            | string   | No       |
| merchantUserCountryCode | query      | Merchant user country code, example PER = PERU                                                                                               | string   | No       |
| merchantUserId          | query      | Merchant user id                                                                                                                             | string   | No       |
| merchantUserEmail       | query      | Merchant user email                                                                                                                          | string   | No       |
| merchantTxId            | query      | Merchant transaction id                                                                                                                      | string   | No       |
| maskedUserAccount       | query      | Masked user account                                                                                                                          | string   | No       |
| ipAddr                  | query      | Ip address                                                                                                                                   | string   | No       |
| pspRefId                | query      | Psp ref id                                                                                                                                   | string   | No       |
| includeRelated          | query      | Include related transactions                                                                                                                 | boolean  | No       |
| txDateFrom              | query      | Transaction creation date from yyyy-MM-dd HH:mm:ss                                                                                           | dateTime | No       |
| txDateTo                | query      | Transaction creation date to yyyy-MM-dd HH:mm:ss                                                                                             | dateTime | No       |
| txLastUpdatedDateFrom   | query      | Transaction last updated date from yyyy-MM-dd HH:mm:ss                                                                                       | dateTime | No       |
| txLastUpdatedDateTo     | query      | Transaction last updated date to yyyy-MM-dd HH:mm:ss                                                                                         | dateTime | No       |
